## Getting Started

Follow these steps to set up and run the project locally:

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   ```
2. **Navigate to the project directory:**
   ```bash
   cd 2026-frontend-challenge-izmir
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Environment Variables:**
   Create a `.env` file in the root directory and add the required environment variables (e.g., `VITE_API_KEY`, `VITE_FORM_ID_...`).
for example:
```env
VITE_API_KEY=your_api_key_here
VITE_FORM_ID_CHECKINS=
VITE_FORM_ID_MESSAGES=
VITE_FORM_ID_SIGHTINGS=
VITE_FORM_ID_PERSONAL_NOTES=
VITE_FORM_ID_ANONYMOUS_TIPS=
 ```
5. **Start the development server:**
   ```bash
   npm run dev
   ```

The application will be running at `http://localhost:5173/`.
